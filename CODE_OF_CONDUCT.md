# Code of Conduct

This project adopts a professional, respectful, and inclusive collaboration model.

## Expected behavior

- Be respectful and constructive.
- Focus on technical issues, not personal attacks.
- Provide reproducible reports and evidence.

## Unacceptable behavior

- Harassment, discrimination, or abusive language.
- Deliberate disruption of project collaboration.

## Enforcement

Maintainers may remove or moderate content and participation that violates this policy.
