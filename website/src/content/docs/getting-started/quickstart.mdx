## Prerequisites

| Tool         | Minimum version   | Purpose                   |
| ------------ | ----------------- | ------------------------- |
| GCC or Clang | gcc 12 / clang 16 | C11 cross-compilation     |
| CMake        | 3.22              | Build system              |
| Ninja        | 1.10              | Build backend             |
| QEMU         | 8.0               | arm64 smoke validation    |
| Python       | 3.10              | Config validation scripts |

<Tabs>
    <TabItem label="macOS (OrbStack)">

    <TabItem label="Ubuntu / Debian">

        ```sh
            qemu-system-aarch64 python3 cmake ninja-build
        ```
    </TabItem>

## Clone and Build

**1. Clone the repository**

```sh
git clone https://github.com/NguyenTrongPhuc552003/haven.git
cd haven
```

**2. Install git hooks** (optional but recommended)

```sh
./scripts/setup-hooks.sh
```

**3. Build — host unit + integration tests**

```sh
cmake --preset host-tests
cmake --build build-host
```

Expected output: all test binaries compiled.

**4. Run the full test suite**

```sh
ctest --test-dir build-host --output-on-failure
```

All tests pass with zero failures.

**5. Run style and config checks**

```sh
cmake --build build-host --target style-check
# or directly:
./scripts/dev/style-check.sh
```

**6. Cross-compile for ARM64 (QEMU)**

```sh
cmake --preset arm64-qemu
cmake --build build
# → build/haven.elf, build/haven.bin, build/guest_a.bin, build/guest_b.bin
```

**7. Run the QEMU smoke test**

```sh
./scripts/qemu-smoke.sh
```

This verifies `qemu-system-aarch64` is reachable. Full QEMU partition boot
is a Release 2 milestone.

## Run the Full CI Preflight Locally

```sh
./scripts/ci/ci-preflight.sh
```

This runs `cmake host-tests → ctest → style-check → check-configs` in sequence,
identical to what the CI pipeline executes.

<Aside type="tip">
    Set `CC=clang` or `CC=gcc` before configuring to match a specific CI matrix leg:
    ```sh
    CC=clang cmake --preset host-tests && cmake --build build-host
    CC=gcc   cmake --preset host-tests && cmake --build build-host
    ```
</Aside>

## Capture a Benchmark Baseline

```sh
python3 scripts/evidence/benchmark-baseline.py
```

Results are written to `build/benchmarks/baseline.json`. Commit this file to
track performance trends over time.
