## System Architecture

```
┌──────────────────────────────────────────────────┐
│               Guest Partitions (EL1/EL0)         │
│   ┌─────────────────┐   ┌──────────────────────┐ │
│   │  Linux (A55×4)  │   │  FreeRTOS (M7 / A55) │ │
│   └────────┬────────┘   └──────────┬───────────┘ │
├────────────┼───────────────────────┼─────────────┤
│            │       EL2 — Haven     │             │
│   ┌────────▼────────────────────────▼───────────┐ │
│   │  Stage-2 MMU │ IRQ Router │ Budget Sched    │ │
│   │  SMMU policy │ EL2 exc handler │ Time acct  │ │
│   └────────────────────────────────────────────┘ │
├──────────────────────────────────────────────────┤
│               Hardware (ARM64 SoC)               │
│   Cortex-A55 × 4  │  GIC-700  │  SMMU-700       │
│   Cortex-M7 (MCU) │  DDR5     │  Peripherals    │
└──────────────────────────────────────────────────┘
```
