## [0.1.0] — 2026-05-10

### Added
- Initial repository structure, governance, and documentation.
- Build/test/style script scaffolding (`build.sh`, `test.sh`, `style-check.sh`).
- CI pipeline: PR build matrix (gcc + clang), nightly validation, benchmark workflow.
- Unit tests: `test_core_stubs`, `test_el2_exceptions`, `test_freertos_integration`,
  `test_guest_uart`, `test_smmu_dma`.
- Integration tests: `test_isolation_flow`, `test_isolation_negative`.
- Static partition configs: `imx95-devkit.yaml`, `qemu-virt.yaml`, `imx8qm-mek.yaml`.
- Config validator (`check-configs.py`).
- Evidence packaging script (`package-evidence.sh`).
- Benchmark baseline script (`benchmark-baseline.py`).
- Chapter traceability matrix, evaluation plan, milestone and release specs.
- i.MX95 bring-up runbook and evidence templates.
- Astro Starlight documentation portal (`website/`).

### Architecture
- Stage-2 MMU stub (`src/core/mm/stage2.c`) — validates structure, no LPAE yet.
- IRQ ownership table (`src/core/irq/ownership.c`) — full INTID tracking.
- Budget scheduler (`src/core/sched/budget.c`) — accounting without timer integration.
- SMMU stub (`src/core/dma/smmu.c`) — stream ID tracking.
- EL2 exception handler stub (`src/core/exc/el2_exceptions.c`).
- Guest UART driver (`src/guest/drivers/uart.c`).
- FreeRTOS integration layer (`src/guest/rtos/freertos_integration.c`).
